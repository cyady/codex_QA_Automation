# 동적 이메일 수신자 그룹 테스트케이스

## 범위/목적
- 기준 가이드: https://guide.recatch.cc/ko/articles/%ED%95%84%ED%84%B0-03e6d75a
- 필드 유형 커버:
  - 텍스트, 긴텍스트
  - 날짜, 숫자, 백분율, 통화, 이메일, 전화번호
  - 단일 선택 목록, 다중 선택 목록
  - URL
  - 사용자
  - 체크박스
- 실제 사용자 시나리오(고액/저액, 중복 이메일, 회사 선택, 수신 동의 등)를 함께 검증

## 공통 전제
1. `lead_seed.csv` 기반 10개 리드가 생성되어 있어야 한다.
2. 자동화 생성 title은 `원본title-CASE_ID-타임스탬프` 형태이므로, title 필터는 `contains QA_DYN_00x`로 검증한다.
3. 자동화로 안정 커버 가능한 필드는 아래 4개다.
   - `title`
   - `email` (`QA-이메일` / `테스트(이메일)`)
   - `website` (`QA-URL` / `테스트(도메인)`)
   - `amount_krw` (`금액`)
4. 연락처 선택은 이번 테스트에서 비워둔다.

## 수동 보강(1회) - 자동화 미커버 필드용
- 아래 값은 필드 유형 테스트를 위해 웹에서 드로어/모달에서 직접 수정한다.
- 필드 라벨은 환경에 맞춰 매핑(예: `커스텀 날짜`, `테스트(숫자, 소수점 자릿수 4)`, `테스트(백분율...)`, `테스트(선택)`, `옵션 여러개`, `사용자`, `테스트(체크박스)`, 긴텍스트 필드).

| case_id | 텍스트 | 긴텍스트 | 날짜 | 숫자 | 백분율 | 전화번호 | 단일선택 | 다중선택 | 사용자 | 체크박스 |
|---|---|---|---|---:|---:|---|---|---|---|---|
| QA_DYN_001 | TEXT_ALPHA | LONG_ALPHA_ENTERPRISE | 2026-02-15 | 42.1234 | 12.5000 | 01070000001 | QA_SINGLE_A | QA_MULTI_A,QA_MULTI_B | 나(테스터) | true |
| QA_DYN_003 | TEXT_BETA | LONG_BETA_LOW_BUDGET | 2026-01-20 | 5.0000 | 2.0000 | 01070000003 | QA_SINGLE_B | QA_MULTI_B | 다른 팀원 | false |
| QA_DYN_008 | TEXT_GAMMA | LONG_GAMMA_OPTOUT | 2026-02-18 | 88.0000 | 33.3300 | 01070000008 | QA_SINGLE_A | QA_MULTI_C | 나(테스터) | true |

주의:
- 단일/다중선택 옵션(`QA_SINGLE_A/B`, `QA_MULTI_A/B/C`)이 없다면 먼저 옵션 추가 후 값 지정.
- 나머지 case는 해당 보강 필드를 비워두거나 기본값 유지.

## 테스트케이스

### 1. title contains 필터 (자동화 커버)
- 테스트 제목: `title` 패턴 필터 검증
- 테스트 방법: `title contains "QA_DYN_001"`
- 기대 결과:
  - 예상 포함: `QA_DYN_001`
  - 예상 제외: `QA_DYN_002~QA_DYN_010`

### 2. 이메일 중복 필터 (자동화 커버)
- 테스트 제목: 중복 email 식별 검증
- 테스트 방법: `email equals "qa_dyn_001@example.com"`
- 기대 결과:
  - 예상 포함: `QA_DYN_001`, `QA_DYN_006`
  - 예상 제외: 나머지 8개

### 3. URL exact/contains 필터 (자동화 커버)
- 테스트 제목: URL 필터 검증
- 테스트 방법: `website contains "legacy-sys.example.com"`
- 기대 결과:
  - 예상 포함: `QA_DYN_010`
  - 예상 제외: 나머지 9개

### 4. 통화 금액 임계값 필터 (자동화 커버)
- 테스트 제목: `amount_krw` 임계값 검증
- 테스트 방법: `amount_krw >= 1000000`
- 기대 결과:
  - 예상 포함: `QA_DYN_001`, `QA_DYN_002`, `QA_DYN_004`, `QA_DYN_006`, `QA_DYN_007`, `QA_DYN_008`, `QA_DYN_009`
  - 예상 제외: `QA_DYN_003`, `QA_DYN_005`, `QA_DYN_010`

### 5. 통화 구간 필터 (자동화 커버)
- 테스트 제목: 금액 구간 필터 검증

## 참고사항
1. 자동화 커버 가능:
   - `title`, `email`, `website`, `amount_krw`
   - 위 필드만 사용하는 테스트는 자동화만으로 회귀 가능
2. 자동화 미커버 필드(텍스트/긴텍스트/날짜/숫자/백분율/전화/단일·다중선택/사용자/체크박스)는 웹에서 수동 수정 후 필터 검증 진행
3. 연락처 선택은 본 문서 범위에서 비워두고, 이메일 필드로 대체 검증
