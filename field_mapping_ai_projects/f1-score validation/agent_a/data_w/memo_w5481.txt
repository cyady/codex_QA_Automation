의제





SL 브라질 공장에 AMR을 적용한 물류 자동화가 1차적으로 검토 중



기존 컨베이어 중심 이송 방식을 축소 또는 제거 → AMR 기반 직접 이송 가능성 검토



단기 적용 대상은 물류/조립



중,장기적으로는 렌즈 사출 → 하드코팅 공정 자동 이송까지 확대 검토



사출 라인은 2028~2029년 신규 대형 사출기 도입 예정 (순차적 2대)

미팅 내용

1. 기존 운용 방안 (이슈사항)





컨베이어 방식





설치 공간 과다 점유



대형 사출기 + 커브 컨베이어로 레이아웃 복잡



렌즈 특성상





정전기 발생



파티클(이물) 유입에 매우 민감



도장(하드코팅) 공정 불량 리스크 큼

2. 사출 → 하드코팅 AMR 적용 개념





제품: 렌즈 사출품 LH+RH 1세트 / 최대 1.5kg



사출 C/T: 65~72초



이송 거리: 약 10m



운영: 24시간, 사출 직후 즉시 단품 이송 필수

3. AMR 기본 역할





사출기에서 렌즈 직접 적재



하드코팅룸 입구로 이동



도킹 후 룸 내부 컨베이어로 밀어넣기



빈 상태로 사출 위치 복귀



기대 효과





외부 컨베이어 최소화



공간 절약



기존 Nepler와 연동에 우수 

4. 핵심 기술 이슈





 제전·파티클 관리 (가장 중요)





렌즈 도장 공정 특성상 이물 민감



AMR 상부 커버 + 제전 에어 블로잉 필수



단순 커버 + 다수 적재 방식 ❌



하드코팅룸 투입 방식 (검토 중)





안 1: AMR이 사출품만 이송 → 컨베이어 도킹



안 2: 렌즈 + 지그 일체형 이송

5. 공정 환경





사출 공정: 외부 노출 / Class 100,000~3,000,000



하드코팅 공정: 





내부 컨베이어



입구 제전 / 출구 제전 + 에어블로잉



AMR 구간에서 청정·제전 보완 필수

향후 일정





제작 가능 여부 금주 중 답변 요망
