간이버퍼 사양변경



요약





간이버퍼 수량 변동 없음



1단-->2단 혼합형 (상층 : 무동력 / 하층 : 무동력)



상층 역할 : 무동력 롤러 방식 / 완제품 임시 적재용



하층 역할 : 동력 컨베이어 방식 / 트레이 자동 배출



적재 용량 : 5개 트레이 (Key 라인 규격 - 500*350*30 mm, 2.3kg / SSB라인 규격 - 520*360*85 mm, 2.6kg)



자동화 범위 : 빈 트레이 포함 완전 자동화 모색



상세 내용

1. 구조 변경





기존 1단 구조에서 2단(2레벨) 구조로 변경 요청.



상층과 하층의 기능을 분리하여 작업 효율성을 높이고, 향후 자동화 확장에도 대응할 수 있도록 설계 필요.

2. 상층 사양





상층은 무동력 롤러 방식으로 구성 요청.



로봇(컨베이어형 AMR 등)이 트레이를 밀어 넣는 방식으로 작동.



완제품 또는 적재 대기 트레이 보관용으로 활용.

3. 하층(아래층) 사양





하층은 동력형 컨베이어 구조로 구성.



작업이 완료된 트레이를 자동으로 배출.



로봇 또는 AMR과 도킹되어 트레이를 자동 이송할 수 있어야 함.



구동부 및 제어 시스템 포함이 필수임.

4. 동력·무동력 혼합 구조





상층은 무동력, 하층은 동력형으로 구성하는 혼합형 간이버퍼 구조 요청.



이는 상층에서는 수동 적재(로봇 밀입력) 기능을, 하층에서는 자동 배출 기능을 동시에 구현하기 위함.



이 구조가 기술적으로 구현 가능한지, 제어 로직 구성 및 안전성 확보 측면에서 공급사 기술팀의 검토가 필요할 것으로 전망.

5. 트레이 적재 가능 수량





상층(무동력부)은 여러 개 트레이 적재가 가능한 구조를 요구.



기본적으로 2~3개 이상, 가능하다면 최대 5개 트레이 적재 가능하도록 검토 요청.



적재 시 높이 간섭, 무게 하중, 센서 감지 안정성 등에 대한 기술적 확인 요망.



6. 수량 및 적용 라인





간이버퍼는 총 7대로 고정.



우선 적용 대상은 키라인(주 생산 라인)으로 확정.



7. 자동화 적용 범위





기존의 단순 버퍼 기능을 넘어, 빈 트레이 회수 및 공급까지 포함하는 완전 자동화 시스템 구상.



향후 추가 AMR(리프트형 등) 도입 시에도 높이 차이를 조정할 수 있도록 구조적 확장성과 연동성을 확보해야 함.

설계 및 견적 반영 시 유의사항







2단 구조(상층 무동력 / 하층 동력) 명확히 구분하여 견적 반영할 것



동력·무동력 혼합 구조의 제어 및 구동 가능 여부 기술 검토 필요



트레이 최대 5개 적재 가능 여부 확인 및 구조 보강 시 비용 반영



커스터마이즈 소요기간 및 추가비용 별도 표기



향후 리프트형 전환 및 AMR 연동 가능 구조로 설계 여유 확보
