 ▶ 비대면 미팅 (Zoom 미팅) - 하이웍스 전자결재 회의록 내용 참고
 1. 자동차 필터 샘플에 트윈위즈의 항균 제품 적용
 ▶ 프리미엄 라인에 적용 예정. 필터 샘플과 필터 원단 샘플 공유. 
 ▶ 먼지 제거층과 지지층으로 구성되어 있으며, 먼지 제거층은 멜트 블로운 기법을 활용 (정전기 처리), 지지층은 나노 코팅 처리 (기존 산화아연을 사용한 것 추측)
 ▶ 뉴라이즌 제공 샘플을 트윈온에 침지 방식으로 적용 후 평가 → 물성 평가 : 뉴라이즌/ 항균평가 : 트윈위즈
 2. 필요 인증 내용 공유 및 준비사항 전달
 ▶ 트윈온을 적용한 제품 개발 완료 목표 시점은 2026년 상반기 출시 되는 차량. → 해당 기간에 제품 제조 및 판매에 문제 없다 답변. 살생물제 인증 관련 상세 내용은 메일로 전달 예정.
 3. 침지 방식의 성능 저하와 대안 방안
 ▶ 침지 방식으로 원단 처리 시 집진성능이 저하될 우려 有.
  → 자체적으로 판단하였을 때 성능 저하는 없을 것으로 판단됨.
 ▶ 침지 방식 외 스프레이 코팅 방식으로 적용 예정.
 ▶ 압력 손실의 경우 트윈온 적용 후 완전 건조 시에 문제 없을 것으로 판단.
 4. 기타
 ▶ 항균 기능 부여로 인한 필터 원단 원가 상승률 우려 → 추후 논의 사항으로 제외. 현재는 기능 발현을 우선 평가.
 ▶ 탈취 기능 가능 여부 문의. 검증 필요하다 전달. → 현재 활성탄을 많이 사용. 탈취 평가 물질 공유 예정.
 ▶ 추가 지지체 원단 샘플 (5m) 제공 예정.
