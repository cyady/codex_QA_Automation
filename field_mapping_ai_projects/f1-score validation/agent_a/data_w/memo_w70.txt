AMR 시나리오
총 4가지 시나리오를 구성중이셨습니다.
SMT 2번 line → Test line 까지의 매거진 운송
OVEN → Test 공간으로 자동문을 통과하는 트레이 운송
Packing → 출하로 자동문을 통과하는 매거진 운송
SMT 7번 line → Test line 까지의 매거진 운송
첫번째로 도입가능성이 높은 것은 OVEN → Test 공간 이라고 말씀주셨습니다.
신봉걸 책임님께서 해당 라인의 실무 반장님께 여쭤보실 예정
