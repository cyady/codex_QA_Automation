미팅 내용

1. 기업 개요





연매출 : 5779억원 (영업이익 : -23억)



직원 수 : 185명



주소 : 경기도 평택시 팽성읍 추팔산단로 63



사업 분야 : 자동차용 전장부품





주요 생산품 : 차량 대시보드용 메인보드 (Front), Car Audio 메인보드, Key Panel (볼륨/공조 등 인터페이스에 쓰임), DAB/DMB 수신 모듈, GPS 모듈, Telematics 제품 등.

2. 배경





제조/물류 공정 내 AMR 도입 검토



타입별 비용 확인 요망





미팅 이후 전달 예정

3. 운용 환경





적용 구간 : SMT라인



공장 위치 : 평택 공장 2개





SMT 라인 : 총 30개 (자세한 사항은 아래 "4. 공정개요," 참조)

4. 공정 개요





SMT 공정: PCB에 부품을 실장하고 솔더링으로 고정



후공정: SMT로 어려운 작업(자삽/수삽, 라우트, 에폭시, 하네스 체결 등) 및 검사/검증(X-RAY, ICT, 에이징, 기능검사 등) 수행





후공정은 자동화 설비 기반으로 정밀 및 고속 처리 가능, 고객 요구에 따라 세부 공정 분화



SMT 공정 구성(흐름)





로더 → 레이저마킹 → 스크린프린터(솔더 도포) → SPI 검사 → 실장기(칩 부품) → MAOI 검사 → 이형실장기 → 경화/리플로우 → AOI 검사 → 언로더(매거진 적재)
