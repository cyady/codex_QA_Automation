주문/발송 
지사 담당자 계정으로 주문 발송 (지사 5~6개 예상)
발송 전 퇴사자 발생 시 → 부분 취소 처리 (예약건 핀 취소)
결제 방식별 취소 처리
카드 결제 → 부분 취소 가능
입금 결제 → 환불 처리
취소 건 발생 시 소통 채널로 요청 → 즉시 처리 (최대 1시간 이내)
소통 채널 협의 필요: 오픈 카톡, 구글 스프레드시트 등

지사 담당자 계정 관리
특정 상품만 노출 원함 -> 불가 안내
특정 상품 제외 가능 (현금성 상품 금액 변경으로 제한 ex.999,999 )
일반회원 으로 관리 → 상품권류 구매 불가
단, 특정 상품만 일반회원도 구매할 수 있도록 설정 필요
추후 더 적합한 관리 방식 모색 예정

업체 준비 사항
진행 확정 상품 리스트 전달 필요

센드비 제공사항
상품 리스트 전달받으면 최종 견적/할인율 안내
총 관리자가 확인 가능한 개별 계정 생성 및 로그인 정보 전달 (원하시는 경우)
관리자 직접 주문내역 확인 가능
요청 시 센드비에서 주문내역 전달 필요

정산 관련
이체 거래 시 전자 거래명세서 / 견적서 제공 필요
개인정보 파기 확인서 제공 필요
