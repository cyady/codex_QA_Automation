㈜하티웍스 재무건전성 평가 (2022~2024)

1. 성장성
매출액
2022: 약 7.9억 원
2023: 약 12.7억 원
2024: 약 16.4억 원
 → **연평균 성장률(CAGR) 약 45%**로 매우 양호
자산
2022: 51억 원 → 2023: 82억 원 → 2024: 99억 원
 → 꾸준한 증가, 특히 유동자산이 두 배 이상 확대
✅ 강점: 소규모지만 빠른 성장 궤도

2. 수익성
영업이익
2022: 0.98억 → 2023: 2.15억 → 2024: 0.77억
 → 2024년 급락 (이익률 17% → 4.7%)
순이익
2022: 0.98억 → 2023: 3.32억 → 2024: 1.61억
 → 흑자 유지했지만, 2023 대비 절반 이하 감소
수익성 지표
영업이익률: 12.4%(22) → 17.0%(23) → 4.7%(24)
순이익률: 12.5%(22) → 26.2%(23) → 9.8%(24)
ROE: 83.5%(22) → 99.8%(23) → 27.7%(24)
⚠️ 리스크: 매출은 커졌지만 영업 효율성이 떨어짐 → 수익성 둔화 뚜렷

3. 안정성
부채비율: 207.8%(22) → 64.8%(23) → 50.6%(24) (안정적)
자기자본비율: 32.5%(22) → 60.7%(23) → 66.4%(24)
차입금 의존도: 39.7%(22) → 19.5%(23) → 16.1%(24)
이자보상배수: 13.8배(22) → 25.3배(23) → 9.1배(24) (여전히 충분)
✅ 안정성은 대폭 개선, 부채 부담 거의 없음

4. 현금흐름
영업활동 현금흐름
2022: +1.8억 → 2023: +27억 → 2024: +0.5억 (급감)
투자활동 현금흐름
2024: -0.6억 (설비·투자 집행)
현금영업이익: 2024년 -0.16억 (적자 전환)
⚠️ 경고 신호: 이익은 흑자지만 실제 현금 창출력 약화

5. 정성 지표
수상/PR: 국무총리상, 고용노동부 장관 표창, 사회적 가치기업으로 언론 다수 언급
사업 성격: 장애인 고용 + AI 솔루션 등 ESG/임팩트 기반
글로벌 확장: 2024년 중국 청도 법인 설립
채용/인원: 꾸준한 채용 진행, 팀 확장 신호

6. 종합 평가 (100점 만점)
투자·시장: 20/35 (투자 유치 데이터 부재, 대신 글로벌 확장으로 보완)
재무 지표: 28/35 (매출 성장·자본 건전성 긍정, 수익성 둔화로 감점)
정성 지표: 18/20 (수상·레퍼런스·ESG 기업 이미지 우수)
보조 지표: 10/10 (채용·시장 확장·브랜드 신뢰도)
 → 총점: 76/100

7. 결론
판단: 강소기업 (초대 권장)
포지션: 매출 규모는 아직 소기업 수준이지만, 안정적 자본구조 + 지속 성장성 + 사회적 레퍼런스로 강소기업 ICP에 부합
리스크: 영업이익률 급락 & 현금흐름 둔화 → 구매력 측정 시 단기적 주의 필요
권고안:
대표/경영진급 초대는 타당
다만, 재무건전성은 안정적이지만 구매력은 중간 수준으로 평가하는 것이 적절
