1. 영업 현황





기존: 인바운드 중심(과거 사용 경험 있는 유저가 기관/기업에서 도입 요청)



변화: 인바운드 유입 감소(21~23년 지속 감소) → 아웃바운드 시도 시작



타깃:



 1순위 대기업 → 2순위 코스피 상장사 → 3순위 코스닥 상장사



 연구소·R&D 부서 보유 기업 우선



영업 풀: 총 400여 개, 현재 시도 중 약 50개



DB 확보: 홈페이지·대표메일 기반, 수작업 수집

2. 영업/재계약 관리 방식





ERP(비즈):



영업 히스토리(통화·미팅·이메일) 수기 기록



다음 액션 일정 알림(메일로 수신)



세금계산서 발행 및 계약 관리 가능



시트(엑셀):



재계약 일정·담당자·컨택포인트 관리



비즈에서 보기 불편한 정보 필터링·정리용



문제점:



비즈·시트·팀내 공유파일 3중 관리 → 중복 입력·누락 발생



재계약 시즌(연말·연초) 업무 폭증(100~200건/월)



메일 개별 발송, 벌크 발송 어려움(기관별 맞춤 내용 필요)



필드 간 연산 + 이메일 내 변수 확장하면 해결 가능할 것으로 보임

3. 마케팅/리드 관리





리드 유입: 웹 콘텐츠, 세미나/웨비나 가능성 탐색 중



폼 현황:



홈페이지 기본 폼(대리점도 사용) → 커스터마이징 어려움



리캐치 폼 사용 계획(아웃바운드 신규 리드 중심)



리드 관리 과제:



UTM 정보 유지 여부



기존 폼과 리캐치 폼 통합 가능성



랜딩페이지 제작 리소스 부족 -> 그로스 패키지 가능성 열려있음

4. 실무 불편·개선 니즈





반복 작업·중복 입력 줄이기(비즈시트 통합 또는 자동화)



대량 메일 발송 시 기관별 맞춤 정보 자동 반영 기능



리드 소스·여정·성과 추적(UTM, 콘텐츠 반응, 전환율)



재계약 알림·팔로업 자동화



아웃바운드 영업 활용 콘텐츠 필요

5. 리캐치가 제안한 지원 방안





통합 리드 관리: 리캐치 폼·기존 폼 연동, 대시보드에서 통합 조회



알림 자동화: 계약만료일 등 특정 필드 기반 슬랙 알림



데이터 필드 커스터마이징: 기관명·등급·컨택포인트·견적 등



B2B 콘텐츠 전략:



퍼널(인지→관심→고려→구매)별 콘텐츠 운영



산업별·고객사 성공사례 중심 제작 → 아웃바운드 전환율 향상



랜딩페이지/원페이저 제작 패키지 제공



마케팅-영업 협업: 콘텐츠와 콜드메일 결합, 성과 추적·분석
