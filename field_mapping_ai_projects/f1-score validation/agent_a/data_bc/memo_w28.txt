1/ Commit level: 2





실무진(김준성, 오예나 매니저)의 니즈는 매우 명확함.





현재 단순히 리드를 수집하는 단계를 넘어, 실제 영업 기회로 전환(물동량 확보)하기 위한 너처링 및 성과 측정(대시보드)에 대한 갈증이 큼.



구체적인 기능(UTM, 고객 여정, IP 보안 이슈)과 견적에 대해 상세히 질문하며 적극적인 검토 의사를 보임.





다만, 최종 의사결정권자가 아니며, 기존 영업팀이 주도하는 파이프드라이브(PipeDrive) 체계가 있어 내부 설득 과정이 필요함.



2/ 예상 금액
약 1,000만 원 ~ 1,300만 원 (연간, VAT 별도)





기본 구성 (CRM + 온보딩): 약 1,020만 원 (CRM 연간 구독료 약 624만 원 + 컨시어즈 온보딩 400만 원).



파이프드라이브 연동 포함 시: 약 1,300만 원 (외부 솔루션 연동 플랜 약 900만 원 + 온보딩 400만 원).





참고: 별도의 너처링/콘텐츠 컨설팅 진행 시 약 2,400만 원 추가 발생 가능하나 필수 사항은 아님.



3/ Next action





고객사 (두핸즈):





참석자 오예나 매니저 이메일 주소 공유.



자사 웹사이트의 IP 보안/차단 관련 구체적인 제약 사항 정리하여 전달.



(핵심) 3월 중순 이후 의사결정권자(임원 등)가 참석하는 오프라인 미팅 자리 마련.



공급사 (리캐치):





전달받은 보안 이슈(IP 차단 등)와 관련하여 고객 여정 데이터 수집 가능 여부 기술팀 확인 후 공유.



3월 중순 이후 오프라인 미팅 참석 및 데모 시연.



4/ BUILD Framework





Blocker (장벽):





기술적 제약: 내부 보안 정책으로 인한 IP 데이터 수집 불가 가능성.



비용: 현재 사용 중인 파이프드라이브(연 700만 원 수준) 대비 높은 도입 비용.



조직 역학: 영업팀이 이미 파이프드라이브를 주도적으로 사용 중이며, 데이터 주도권이 영업팀에 있음.



User (실사용자):





마케팅팀 김준성 매니저, 오예나 매니저.



Influencer (영향력 행사자):





리멤버 고성호 매니저 소개



Leader (의사결정권자):





고은진 이사 (마케팅 총괄).



원다솜 총괄 등 상위 임원진 (이전 미팅 이력 존재)이 또 의사결정에 영향을 주는 지는 파악 필요



Driver (추진자):





김준성 매니저 (마케팅 성과 개선 및 너처링 필요성을 느끼고 문의함).



5/ BANT Framework





Budget (예산):





구체적인 예산이 확정된 상태는 아니며 정보를 알아보는 단계임.



기존 툴보다 높은 비용(1,000만 원 이상)에 대한 내부 설득(임원 보고)이 필요함.



Authority (권한):





미팅 참석자(김준성, 오예나)는 실무 매니저로 최종 결정 권한은 없음.



고은진 이사 및 상위 임원진의 결재가 필요함.



Need (필요성):





높음. 메타 광고 등을 통해 리드는 확보했으나 영업 기회(물동량)로 전환되지 않는 문제 해결 시급.



스프레드시트 수작업 관리의 한계를 느끼고 있으며, 마케팅 성과를 증명할 대시보드와 고객 여정 데이터 파악이 필요함.



Timing (시기):





현재 탐색 단계이며, 3월 중순 이후 오프라인 미팅을 통해 의사결정권자를 설득한 후 도입 여부가 결정될 것으로 보임.
